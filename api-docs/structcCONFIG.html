Field Documentation

◆ blocks_fragmentation_threshold

double cCONFIG::blocks_fragmentation_threshold

Treshold for blocks overusage by partially filled blocks.

blocks_fragmentation_threshold is obtained as partial used blocks count divided by minimum needed fully filled blocks amount.

◆ cache_size

size_t cCONFIG::cache_size

Blocks cache size in blocks.

if cache_size is set to 0, then blocks cache if disabled.

◆ codec_name

char* cCONFIG::codec_name

compression/decompression codec name.

codec_name should be a valid SquashCodec name or a custom C-style string
If a "custom" value is set then user is to provide an additional three functions:

  • compress
  • uncompress
  • get_max_compressed_size

◆ flash_method

enum cFLASH_MODE cCONFIG::flash_method

Strategy of writing blocks to storage.

Free blocks storing strategy which determines whether should storage file be optimized for filesystem supporing sparce files or not.

◆ fragmentation_threshold

double cCONFIG::fragmentation_threshold

Treshold for space overusage by free blocks.

fragmentation_threshold is obtained as all free blocks size divided by all compressed blocks size.
